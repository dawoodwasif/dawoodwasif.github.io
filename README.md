You can visit the live version at [dawoodwasif.github.io](https://dawoodwasif.github.io).

## 🛠️ Project Structure
- `index.html`: The main landing page.
- `stylesheet.css`: Custom styling for the website.
- `images/`: Directory containing project thumbnails and profile photo.
- `data/`: Directory containing the resume/CV and other public documents.
